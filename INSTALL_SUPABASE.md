# Install Supabase

To complete the Supabase integration, you need to install the Supabase client library:

```bash
npm install @supabase/supabase-js
```

After installation, the following will work:
- ✅ Saving activities per user to Supabase
- ✅ Fetching activities per user from Supabase
- ✅ User-isolated activity logs

Run this command and then test the application!
